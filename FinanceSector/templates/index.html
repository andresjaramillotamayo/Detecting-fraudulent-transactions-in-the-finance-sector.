Modelo

La detección del fraude es una técnica bastante compleja ya que este se comporta como una anomalía, por ello, se presenta una alta descompensación de clases. Para este caso se hizo uso de Light GBM (Gradient Boosting Machine), este marco de trabajo se basa en algoritmos de árboles de decisión, el cuál se utiliza para clasificar y otras tareas de aprendizaje automático.
